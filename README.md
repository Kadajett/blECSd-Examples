# blECSd Examples

Example projects for the [blECSd](https://github.com/Kadajett/blECSd) terminal UI library.

## Templates

Each directory is a standalone example project. To use one:

1. Copy the directory
2. Run `npm install` (or pnpm/yarn)
3. Run `npm run dev`
