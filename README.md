# blECSd Examples

Example projects for the [blECSd](https://github.com/Kadajett/blECSd) terminal UI library.

Each directory is a standalone example project. To use one:

1. Copy the directory
2. Run `npm install` (or pnpm/yarn)
3. Run `npm run dev`

## Applications

Full applications built with blECSd, demonstrating real-world usage patterns.

### File Manager

A dual-pane file manager with tabbed navigation, file previews, and keyboard-driven controls.

![fileManager](https://github.com/user-attachments/assets/f2e4daae-9009-47f3-8c93-c2b299b583e9)
